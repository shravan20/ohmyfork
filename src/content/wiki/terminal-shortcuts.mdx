# Terminal Shortcuts & Commands

## Navigation Shortcuts

- Ctrl + A - Move to beginning of line
- Ctrl + E - Move to end of line

## File Operations

```bash
ls -la      # Detailed list
cd ..       # Parent directory
mkdir -p dir
```
